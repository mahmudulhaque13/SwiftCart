1. What is the difference between null and undefined?
   undefined = এখনও মান পাইনি

null = ইচ্ছা করে খালি রাখা হয়েছে

2. What is the use of the map() function in JavaScript? How is it different from forEach()?
   map() নতুন array বানায়
   forEach() শুধু loop চালায় (কিছু return করে না)

3. What is the difference between == and ===?
   == value compare করে (type ignore করে)
   === value + type দুটোই compare করে

4. What is the significance of async/await in fetching API data?
   এটা asynchronous কাজকে synchronous এর মতো সহজ করে।
   .then() chain লাগে না
   কোড পড়তে সহজ
   error handle করা সহজ

5. Explain the concept of Scope in JavaScript (Global, Function, Block).
   Global Scope => সব জায়গা থেকে ব্যবহার করা যায়
   Function Scope => শুধু function এর ভিতরে ব্যবহার করা যায়
   Block Scope => শুধু block{} এর ভিতরে ব্যবহার করা যায়
